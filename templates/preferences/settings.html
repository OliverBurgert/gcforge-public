{# ----- Database Maintenance ----- #}
Database Maintenance

VACUUM rewrites the database file, reclaiming space left by deleted and updated rows. It is safe to run at any time — if interrupted, the original database is untouched. While running, other database operations are queued (typically a few seconds to a minute). GCForge runs this automatically at startup when free space exceeds 10% of total size and 50 MB.
