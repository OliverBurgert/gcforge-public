About GCForge

GCForge is an open-source geocache management tool — a spiritual successor to GSAK.

Third-Party Icon Attributions
c:geo Icons — Apache License 2.0

Cache type and attribute icons are derived from the c:geo open-source Android geocaching app, licensed under the Apache License 2.0.

Cache type icons

Converted from Android Vector Drawable format. All cache type icons are original geometric artwork created by the c:geo team — not copies of Groundspeak designs.

Attribute icons

Attribute icons originate from multiple sources with individual licenses. The c:geo project maintains a comprehensive attribution list in main/project/attributes/iconlist.txt.

c:geo contributors (self-made icons)

Several attribute icons were created or substantially reworked by c:geo developers: koem (available, fee, flashlight, frontyard, hike variants, hunting, poisonoak, ticks, UV, and others), rsudev (survey marker, offset cache, safari cache), bekuno (munzee, trackables).

Custom placeholder icons

Placeholder icons for cache types not covered by c:geo (Benchmark, Drive-In, Math/Physics, Moving, Own, Podcast) were created for GCForge as simple geometric outlines.
