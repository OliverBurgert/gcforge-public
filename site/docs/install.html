Installation

Platform-specific instructions for installing GCForge.

System Requirements

  • OS: Windows 10+, macOS 12+, or Linux (Ubuntu 22.04+, Fedora 38+)
  • RAM: 2 GB minimum (4 GB recommended for large collections)
  • Disk: 200 MB for the application + space for your cache database
  • Other: No Python or other dependencies required — everything is bundled

Windows

  1. Download the installer from GitHub Releases (gcforge-setup-x64.exe)
  2. Run the installer — click Next through the wizard
  3. Choose install location (default: C:\Program Files\GCForge)
  4. Optionally create Desktop and Start Menu shortcuts
  5. Click Install and wait for completion
  6. Launch from the Start Menu or Desktop shortcut

Windows Notes

  • Windows SmartScreen may show a warning for unsigned executables. Click More info → Run anyway to proceed.
  • The database is stored in %APPDATA%\gcforge\gcforge.sqlite by default.
  • Uninstall via Settings → Apps → GCForge or the included uninstaller.

macOS

  1. Download the DMG from GitHub Releases (gcforge-macos-arm64.dmg)
  2. Open the DMG file
  3. Drag GCForge.app to the Applications folder
  4. On first launch, right-click the app and select Open to bypass Gatekeeper (or allow in System Settings → Privacy & Security)

macOS Notes

  • Both Intel (x64) and Apple Silicon (arm64) builds are available.
  • The database is stored in ~/Library/Application Support/gcforge/gcforge.sqlite.
  • To uninstall: drag GCForge.app from Applications to Trash, then delete ~/Library/Application Support/gcforge/.

Linux

  1. Download the AppImage from GitHub Releases (gcforge-linux-x86_64.AppImage)
  2. Make it executable: chmod +x gcforge-linux-x86_64.AppImage
  3. Run it: ./gcforge-linux-x86_64.AppImage

Optional: Desktop Integration

To add a desktop entry and menu shortcut:

# Create a desktop entry
mkdir -p ~/.local/share/applications
cat > ~/.local/share/applications/gcforge.desktop <<'EOF'
[Desktop Entry]
Name=GCForge
Comment=Geocache Management Tool
Exec=/path/to/gcforge-linux-x86_64.AppImage
Icon=utilities-terminal
Type=Application
Categories=Utility;
EOF

# Update icon cache
update-desktop-database ~/.local/share/applications/

Linux Notes

  • AppImage runs without installation — no root privileges required.
  • The database is stored in ~/.local/share/gcforge/gcforge.sqlite.
  • If FUSE is not installed (required for AppImage), install it: sudo apt install fuse (Debian/Ubuntu) or sudo dnf install fuse (Fedora).

Building from Source

For developers who want to run GCForge from source: 

# Clone the repository
git clone https://github.com/OliverBurgert/gcforge-public.git
cd gcforge

# Install dependencies with uv
uv sync

# Run the development server
uv run python manage.py migrate
uv run python manage.py runserver 127.0.0.1:8000

See the project README for development setup details.
