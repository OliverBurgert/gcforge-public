Getting Started

Install GCForge, import your first caches, and start managing your geocaching life.

Overview

GCForge is a standalone desktop application — no Python installation required. Download the installer for your platform, run it, and you're ready to go.

If you're coming from GSAK, see the GSAK migration guide for step-by-step instructions on importing your existing data.

Quick Start

1. Download & Install

Download the latest beta release for your platform from the download section or directly from GitHub Releases.

See installation instructions for platform-specific details.

2. Launch GCForge

After installation, launch GCForge from your Start Menu (Windows), Applications folder (macOS), or application launcher (Linux). GCForge will open in your default browser.

The first launch creates a new SQLite database in your user data folder. On Windows this is typically %APPDATA%/gcforge/.

3. Import Your First Caches

The quickest way to get started is importing a GPX file from geocaching.com:

  1. Go to Import → GPX / Pocket Query
  2. Select your .gpx or .zip file
  3. Click Import and wait for the background task to complete

GCForge also supports importing from:

  • GSAK databases — select your sqlite.db3 file
  • Adventure Lab exports — GPX files from the lab2gpx tool

See Importing Data for full details.

4. Set Your Location

For distance and bearing calculations, set your home location:

  1. Go to Settings → Reference Points
  2. Add a new reference point with your home coordinates
  3. Set it as your default location

Distance and bearing from your location will appear in the cache list automatically.

5. Explore & Filter

With caches imported, you can:

  • Browse the list — sort by any column, click headers to configure
  • Apply filters — by type, difficulty, terrain, size, status, tags
  • View on map — interactive map with search area drawing
  • Tag caches — organize with many-to-many tags (replaces GSAK databases)
